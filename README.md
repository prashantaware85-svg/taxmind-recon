# TaxMind Recon
Debtor/Creditor Reconciliation SaaS for Chartered Accountants.

## Features
- Exact matching
- Partial matching
- Missing entry detection
- GST reconciliation

## Tech Stack
- Backend: FastAPI + Python
- Database: PostgreSQL
- Container: Docker

## Run Locally
```bash
docker-compose up --build
```

API runs at: http://localhost:8000
